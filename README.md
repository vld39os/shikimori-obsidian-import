# Shikimori Import Obsidian Plugin

Плагин для [Obsidian.md](https://obsidian.md/), формирующий заметки на основе запроса к серверу [Shikimori](https://shikimori.one/).

Для работы плагина требуется аккаунт на сайте [Shikimori](https://shikimori.one/), без него не получится сделать запрос.

# Как использовать

-   Установить плагин в своё хранилище Obsidian (.obsidian/plugins).
-   Включить плагин в настройках (Настройки > Сторонние плагины > Управление плагинами).
-   В боковом меню появится кнопка запуска плагина (аналогично можно вызвать команду в заметке через /).
-   Введите запрос.
-   Выберете удовлетворяющие вашему запросу варианты и нажмите кнопку "Импортировать выбранное".
-   В хранилище создадутся заметки со свойствами.

# В разработке

-   Выбор свойств для запроса.
-   Запрос манге/ранобе.
